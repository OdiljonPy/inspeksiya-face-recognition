# Face Recognition (стройплощадка) — зафиксированные версии под Python 3.14 / RTX 5060 (sm_120, Blackwell)
# Проверено на Windows 11, NVIDIA driver 591.86 (CUDA 13.1 capable).
#
# ВАЖНО про порядок установки (см. README.md):
#   1) torch/torchvision ставятся с ИНДЕКСА PyTorch cu128 (сборка под sm_120):
#        pip install torch==2.11.0+cu128 torchvision==0.26.0+cu128 --index-url https://download.pytorch.org/whl/cu128
#   2) onnxruntime-gpu ставится с extras [cuda,cudnn], чтобы подтянуть CUDA 12 runtime-DLL:
#        pip install "onnxruntime-gpu[cuda,cudnn]==1.26.0" --extra-index-url https://pypi.nvidia.com
#      (отдельный CUDA Toolkit НЕ нужен — DLL приезжают pip-колёсами nvidia-*-cu12)
#   3) остальное — обычной командой: pip install -r requirements.txt
#
# Почему именно так:
#   - Обычный onnxruntime-gpu без CUDA-DLL падает: "cublasLt64_*.dll is missing".
#   - 1.26.0 собран под CUDA 12 и совпадает с CUDA 12.8 у torch — единый рантайм.
#   - sm_120 поддерживается сборкой torch cu128 (arch_list содержит sm_120).
# =============================================================================
